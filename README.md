## 开始

首先运行以下命令进行安装和打包：

> node 版本最好 >= 18.10.0

```bash
# 安装依赖
npm install
# 打包
npm run build
```

然后在项目根目录找到 `public` 文件夹，用浏览器打开 `index.html` 即可，或者[在线浏览](https://zhangzhonghe.github.io/test/public/)。
